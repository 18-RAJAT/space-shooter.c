space-shooter.c [WIP]
=====================
A top-down space shooter written in C with minimal dependencies.

Uses:
- Windowing: Win32
- Rendering: OpenGL
- Audio: XAudio2
- Image Loading: stb_image

Asset Credits
-------------

- music.wav: https://chiphead64.itch.io/free-game-soundtrack-by-cactusdude-hurry-up
