space-shooter.c [WIP]
=====================
A cross-platform, top-down 2D space shooter written in C using only system libraries.

Uses:
- Windowing: Win32 (Windows), X11 (Linux)
- Rendering: OpenGL
- Audio: XAudio2 (Windows), ALSA/pthread (Linux)
- Gamepad: Xinput (Windows), evdev (Linux)
